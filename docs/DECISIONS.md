# Decisions

## Site

- 静的サイトで制作する
- WordPressは使わない
- XSERVERで公開する
- スケジュールはGoogleカレンダーで運用する
- スマホファーストで制作する

## Design

- 背景はやわらかいクリーム色
- メインカラーは深緑
- アクセントは落ち着いた金色
- フォントは Zen Kaku Gothic New
- 余白を大切にする
- 写真は自然な表情を優先する

## Content

- 文章は最後に全面推敲する
- 事実と解釈を分ける
- 実績を盛らない
- 説明しすぎない
- INTERESTSは本棚型・育つページにする

## Brand

- サイトの目的は「知ってもらう」ではなく「会ってみたいと思ってもらう」
- SAUNAページの軸は「楽しく、安全に」
- Project IKURAは伊倉ゆうという表現者のアーカイブとして育てる
